Camel Case always starts out lowercase with each word delimited by a capital letter (like personOne, textUtil, thingsToDo) Pascal Case is similar to camel case, but the first letter is always capitalized (like PersonOne, TextUtil, ThingsToDo) Snake Case means that we delimit words with an underscore (like person_one, text_util, things_to_do)

JavaScript, often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. As of 2022, 98% of websites use JavaScript on the client side for web page behavior, often incorporating third-party libraries.
